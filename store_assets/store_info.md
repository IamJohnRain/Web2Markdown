## 简介 (Short description)

将网页一键转换为 Markdown 并保存到本地

## 详细描述 (Detailed description)

### 中文

Web to Markdown 是一款简洁高效的浏览器扩展，帮助您将任意网页内容转换为 Markdown 格式并保存到本地。

**主要功能：**
- 一键转换网页为 Markdown 格式
- 自选本地磁盘保存位置
- 自动以网页标题命名文件
- 保留标题、段落、列表、链接、图片、代码块、表格等格式
- 支持主流新闻、博客、文档网站

**使用方法：**
1. 浏览任意网页
2. 点击浏览器工具栏中的插件图标
3. 点击「保存为 Markdown」按钮
4. 选择保存位置
5. 文件自动保存，方便后续编辑使用

**适用场景：**
- 保存网页文章用于笔记整理
- 收集技术文档离线阅读
- 备份重要网页内容
- 内容创作者素材收集

### English

Web to Markdown is a simple and efficient browser extension that helps you convert any webpage to Markdown format and save it locally.

**Key Features:**
- One-click conversion of webpages to Markdown
- Choose your own local save location
- Auto-naming with webpage title
- Preserves headings, paragraphs, lists, links, images, code blocks, tables, etc.
- Supports major news, blog, and documentation websites

**How to Use:**
1. Browse any webpage
2. Click the extension icon in browser toolbar
3. Click "Save as Markdown" button
4. Choose save location
5. File saved automatically for easy editing

**Use Cases:**
- Save web articles for note organization
- Collect technical docs for offline reading
- Backup important webpage content
- Content creator material collection

## 隐私政策

此扩展不收集、存储或传输任何用户数据。所有操作均在本地完成。

This extension does not collect, store, or transmit any user data. All operations are performed locally.
