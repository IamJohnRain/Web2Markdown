智能转换

自动提取网页标题和内容，转换为标准 Markdown 格式，保留标题、段落、列表、链接、图片、代码块等所有格式。

使用流程

简单三步，轻松完成网页到 Markdown 的转换

1

点击图标

在任意网页点击浏览器工具栏中的插件图标

2

选择位置

点击保存按钮，选择文件保存位置和名称

3

完成保存

文件自动保存，路径已复制到剪贴板，可直接使用

安装指南

快速安装，即刻开始使用

安装步骤
Step 1: 下载最新版本
前往 Release 页面下载 .zip 压缩包
Step 2: 解压文件
unzip mdPlugin-v1.0.0.zip
Step 3: 打开 Chrome 扩展页面
chrome://extensions/
Step 4: 启用开发者模式
在页面右上角开启「开发者模式」开关
Step 5: 加载扩展
点击「加载已解压的扩展程序」，选择解压后的文件夹
✓ 完成！ 现在您可以在任意网页使用插件了
